# 🎁 Ferns & Petals (FnP) Sales Analytics Dashboard

---

## 📌 Project Overview

This project presents a comprehensive **Sales Analytics Dashboard** built to analyze transactional data from Ferns & Petals (FnP), an e-commerce gifting platform.

The project integrates multiple datasets — **Products, Orders, and Customers** — to deliver business insights across occasions, customer behavior, and product performance.

The objective is to simulate a real-world business intelligence case study by transforming raw relational datasets into actionable insights.

## 📂 Datasets Used

This project integrates **three structured datasets**:

### 1️⃣ Products Dataset
Contains product-level details such as:
- Product ID  
- Product Name  
- Category  
- Occasion  
- Price  

### 2️⃣ Orders Dataset
Contains transactional-level information:
- Order ID  
- Order Date  
- Customer ID  
- Product ID  
- Quantity  
- Revenue  

### 3️⃣ Customers Dataset
Contains customer-related data:
- Customer ID  
- Customer Name  
- City  
- Demographics  

---

## 🔗 Data Modeling

- Established relationships between Orders, Products, and Customers  
- Created a relational data model  
- Performed data transformation and cleaning  
- Built calculated metrics for revenue and performance tracking  

This approach replicates real-world BI workflow used in production environments.

The dashboard highlights:

## 🔍 Key Business Insights

- Certain occasions (e.g., Valentine’s Day, Diwali) drive peak revenue  
- A small group of products contribute majority of sales (Pareto effect)  
- Repeat customers significantly impact total revenue  
- Seasonal demand patterns influence monthly trends  
- Category performance varies across occasions
